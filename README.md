# Sistema Inteligente IRAS — WebApp Premium

Aplicación web premium para analizar el riesgo de afectación a la salud por inseguridad alimentaria en hogares de Lima Metropolitana.

## Módulos
- Inicio / Analizar hogar
- Dashboard ejecutivo
- Mapa de riesgo
- IA Asistente
- Reportes
- Metodología

## Ejecutar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Publicar en Streamlit Cloud
Sigue la guía `GUIA_DEPLOY_PREMIUM.md`.

## Tecnología
- Streamlit
- Scikit-learn
- Random Forest
- Plotly
- ReportLab
- Excel

## Metodología
Proyecto académico de Ciencia de Datos. El dataset es sintético y el IRAS no es diagnóstico médico.
